1. O que é o Sistema Major Leitura ~1 min

O Sistema Major converte dígitos em sons consonantais. Depois, adiciona-se vogais para formar palavras fáceis de lembrar.

Use-o para memorizar números como datas, códigos e listas.

Micro‑check

Resposta correta: A

2. Mapeamento 0–9 (sons por dígito) Tabela rápida
  • 0: z, s, ss, ç, c(e/i)
  • 1: t, d
  • 2: n, nh
  • 3: m
  • 4: r, rr
  • 5: l, ll
  • 6: j, sc, x, ch, g(e/i)
  • 7: q, c, g(a/o/u), qu
  • 8: f, v
  • 9: p, b

Micro‑check

Resposta correta: B

3. Regras e exceções úteis Atalhos

Duplas como ss→0, ll→5, rr→4. "ch" → 6. "g" + e/i → 6; "g" + a/o/u → 7. "c" + e/i → 0.

Micro‑check

Resposta correta: B

4. Exemplos 1–4 dígitos Aplicação
  • 3 → m → "mãe"
  • 32 → m + n → "mina"
  • 327 → m + n + q → "mina que..." (adicione vogais livremente)
  • 3279 → use o conversor e selecione palavras por bloco.

Micro‑check

Resposta correta: B

5. Frases memoráveis Dicas

Prefira imagens vívidas, ação e ligação entre palavras (história curta).

Ritmo e aliteração ajudam a recordar.

Micro‑check

Resposta correta: A

6. Usando o conversor da app Workflow

Digite o número, opcionalmente em blocos com espaços para controlar as partições.

Clique nas palavras para copiar, ou selecione por bloco para combinar no final.

Micro‑check

Resposta correta: A

7. Erros comuns Evite
  • Esquecer que "g" muda (6 vs 7) conforme a vogal seguinte.
  • Forçar palavras longas; prefira curtas e claras.
  • Não testar no conversor quando em dúvida.

Micro‑check

Resposta correta: B
